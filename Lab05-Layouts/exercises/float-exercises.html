Exercise 3 & 4 — Float-Based Layouts

Exercise 3A: Float Image with Text Wrapping

IMG LEFT

This text wraps around the floated image on the left. Floats were originally designed for this exact purpose — wrapping text around images like in a newspaper or magazine. The text flows naturally alongside the image and continues below it once it runs out of space beside it.

This is a second paragraph that continues wrapping around the floated element until the content clears the float height.

IMG RIGHT

This time the image is floated to the RIGHT. Text wraps on the left side. You can see how changing float direction changes which side the text appears on.

This technique is still widely used for article images and blog post layouts.
